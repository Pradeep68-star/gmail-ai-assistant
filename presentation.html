Code: The Time Intelligence Engine

def extract_event_times(text):
    # Inject current year for flexible date formats
    current_year = datetime.now().year
    text_with_year = re.sub(r'\b(\d{1,2}\s+(?:Jan|Feb|Mar|...))(?!\s+\d{4})',
        rf'\1 {current_year}', text, flags=re.IGNORECASE)

    results = search_dates(text_with_year, settings={
        'PREFER_DATES_FROM': 'future',
        'RETURN_AS_TIMEZONE_AWARE': True # Crucial for scheduling
    })
    
    found_times = []
    for _, dt in results:
        dt_ist = dt.astimezone(IST)
        if dt_ist > datetime.now(IST) and dt_ist.time() != datetime.min.time():
            found_times.append(dt_ist)
    
    return sorted(list(set(found_times)))

Key Logic Highlights

  • **Robust Parsing:** Uses the powerful `dateparser.search` library, which handles nearly all natural language date formats, unlike simple regex.
  • **Contextual Clues:** The code adds the current year to incomplete dates (e.g., "Dec 25") to eliminate ambiguity.
  • **Strict Future Filter:** Explicitly checks that the parsed time is **not** in the past and **must** include a specific time (e.g., "5:00 PM," not just "yesterday").
